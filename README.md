-------Liver Cirrhosis Prediction System---------

A Machine Learning-based web application that predicts liver cirrhosis using patient medical data.  
The system uses a **RANDOM FOREST CLASSIFIER** for accurate prediction and analysis.

>>Features
- Liver cirrhosis prediction
- Random Forest ML model
- User-friendly interface

>>Tech Stack
- Python
- Flask
- Scikit-learn
- Pandas
- HTML/CSS
